# Superstore Sales Analytics Dashboard

## Overview
This project presents an interactive Tableau dashboard built using the Superstore dataset. The dashboard provides insights into sales performance, profitability, customer segments, product categories, and regional trends.

## Objectives
- Analyze sales and profit trends over time
- Identify top-performing regions and product categories
- Evaluate sub-category profitability
- Enable interactive analysis through filters

## Tools Used
- Tableau Desktop
- Microsoft Excel
- Data Visualization
- Business Analytics

## Dashboard Features
- KPI Cards (Sales, Profit, Orders, Profit Margin)
- Sales Trend Analysis
- Category and Sub-Category Performance
- Regional Sales Analysis
- Top Products by Sales
- Interactive Filters (Region, Category, Sub-Category, Segment, Order Date)

## Key Insights
- Identified top-performing regions and categories
- Analyzed profitability across product segments
- Detected loss-making products despite high sales
- Explored customer segment contributions to revenue
